# Console-Based ATM Simulation

A functional ATM simulation developed in C# that practices Object-Oriented Programming (OOP) logic and data persistence.

##  Features
* **User Authentication:** Login and Sign-up system.
* **Data Persistence:** User accounts and balances are stored in a JSON file.
* **Interactive UI:** Smooth navigation using arrow keys.
* **Account Management:** Deposit and withdraw money with real-time balance updates.

##  Tech Stack
* **Language:** C#
* **Framework:** .NET
* **Data Format:** JSON (System.Text.Json)

##  How to Run
1. Clone the repository.
2. Open the solution in Visual Studio or VS Code.
3. Run the project.
